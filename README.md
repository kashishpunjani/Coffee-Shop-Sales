# ☕ Coffee Shop Sales Analysis | Excel Dashboard Project

## 📌 Project Overview
This project focuses on analyzing coffee shop sales data (Jan – June 2023) to identify key trends, customer behavior, and business opportunities.

The goal is to transform raw sales data into meaningful insights that can help improve decision-making and overall business performance.

---

## 🎯 Objective
- Analyze sales performance by time (hour, day, month)
- Identify peak sales periods
- Evaluate store-wise performance
- Find top-selling products and categories
- Understand customer ordering behavior

---

## 🛠 Tools & Technologies Used
- Microsoft Excel  
- Power Query (Data Cleaning & Transformation)  
- Pivot Tables & Pivot Charts  
- Data Visualization (Interactive Dashboard)  

---

## 🔍 Data Preparation
- Cleaned and transformed raw data using Power Query  
- Created custom columns (Total Revenue = Unit Price × Quantity)  
- Extracted time-based features (Hour, Day, Month)  
- Categorized product sizes using conditional columns  

---

## 📊 Key Insights
- Peak sales occur between **8 AM – 10 AM**, with 10 AM being the highest  
- **Weekdays generate higher revenue** compared to weekends  
- **Hell’s Kitchen** is the top-performing store location  
- **Coffee category contributes the highest revenue**  
- **Barista Espresso** is the top-selling product  
- Average customer spends **$4.69** and orders **1.44 items**  

---

## 💡 Business Recommendations
- Increase staff during morning peak hours (8–10 AM)  
- Focus on promoting high-revenue products like espresso-based drinks  
- Use upselling strategies to increase average order value  
- Introduce offers/discounts during weekends to boost sales
